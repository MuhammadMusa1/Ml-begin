PANDAS
Работа с таблицами данных
Аналогия Это Excel внутри Python. Только вместо мышки — короткие команды, и таблица в тысячи строк обрабатывается за секунду.
Зачем нужен: вручную тысячи строк не пролистать. Pandas открывает таблицу, считает, фильтрует и сортирует данные одной командой. Главный объект — DataFrame (его называют df).
pd.read_csv(url)
Загрузить таблицу из файла или интернета
df.head()
Показать первые 5 строк — взглянуть на данные
df.tail()
Показать последние 5 строк
df.shape
Размер: (сколько строк, сколько столбцов)
df.info()
Список столбцов, их типы и где есть пропуски
df.describe()
Статистика: среднее, минимум, максимум
df.columns
Названия всех столбцов
df["Age"]
Взять один столбец по имени
df["Age"].mean()
Среднее значение по столбцу
df["Age"].max()
Максимум (и так же .min())
df["Sex"].value_counts()
Сколько раз встречается каждое значение
df[df["Age"] > 18]
Фильтр: только строки, где возраст больше 18
df.sort_values("Age")
Отсортировать по столбцу
df.groupby("Sex").mean()
Сгруппировать и посчитать среднее по группам
df.isnull().sum()
Сколько пропусков в каждом столбце
df.dropna()
Удалить строки с пропусками

NUMPY
Быстрые вычисления над числами
Аналогия Это супер-калькулятор для списков чисел. Одна операция меняет сразу тысячи чисел — не нужно проходить их по одному в цикле.
Зачем нужен: считает быстро и сразу со всем массивом. На нём построены все ML-библиотеки и нейросети — внутри они «думают» массивами NumPy. Главный объект — массив (array).
np.array([1, 2, 3])
Создать массив из списка чисел
np.zeros(5)
Массив из пяти нулей
np.ones(5)
Массив из пяти единиц
np.arange(0, 10)
Числа по порядку: 0,1,2…9
np.linspace(0, 1, 5)
5 чисел равномерно от 0 до 1
a + 1
Прибавить 1 ко ВСЕМ числам сразу
a * 2
Умножить все числа на 2 одной строкой
a.mean()
Среднее значение массива
a.sum()
Сумма всех чисел
a.max()
Максимум (и так же .min())
a.std()
Разброс значений (стандартное отклонение)
a.shape
Размер массива
a.reshape(2, 3)
Поменять форму: например в таблицу 2×3
a[0]
Взять первый элемент (счёт с нуля!)

Оставить только числа больше 5
np.sqrt(a)
Корень из каждого числа сразу

MATPLOTLIB
Графики и визуализация · + Seaborn
Аналогия Это художник для данных. Таблицу чисел превращает в картинку, где закономерность видно глазами за секунду. Seaborn делает то же, но красивее.
Зачем нужен: один хороший график говорит больше, чем сто строк чисел. plt — это matplotlib, sns — seaborn (надстройка для красоты).
plt.figure(figsize=(8,4))
Задать размер будущего графика
sns.histplot(data=df, x="Age")
Гистограмма: «сколько чего» (распределение)
sns.scatterplot(x, y)
Точки: связаны ли два признака
sns.heatmap(df.corr())
Тепловая карта связей всех признаков
sns.boxplot(x, y)
«Ящик»: разброс и выбросы
sns.barplot(x, y)
Столбики для сравнения категорий
sns.lineplot(x, y)
Линия: как меняется со временем
sns.pairplot(df)
Сетка всех scatter сразу одним взглядом
hue="Survived"
Раскрасить точки по третьему признаку
plt.title("Заголовок")
Подписать график сверху
plt.xlabel("Возраст")
Подписать ось X (и plt.ylabel для Y)
plt.legend()
Показать пояснение к цветам
plt.show()
Показать готовый график
plt.savefig("plot.png")
