1
Главный инструмент курса — Google Colab

Хорошая новость: весь код мы пишем прямо в браузере. Не нужно ничего устанавливать на компьютер — ни Python, ни программы. Нужен только аккаунт Google.

Что такое Colab

Google Colab — это бесплатная среда, где можно писать и запускать код Python прямо в браузере. Там уже всё установлено: Python, библиотеки, даже мощные видеокарты для нейросетей.

Открывается по адресу: colab.research.google.com

Что сделать заранее

  1. Завести аккаунт Google (если ещё нет почты Gmail) — на accounts.google.com.
  2. Открыть colab.research.google.com и войти под этим аккаунтом.
  3. Нажать «Создать новый блокнот» и убедиться, что страница открылась.
Если открылся пустой блокнот с полем для кода — всё готово. Это займёт пару минут.
На первом уроке мы сразу запустим в Colab настоящую нейросеть. Никакой установки — просто откроешь ссылку, которую даст преподаватель.
2
Как пользоваться Colab (на 1 минуту)

В Colab код живёт в «ячейках». Ты пишешь команду в ячейку и нажимаешь на кнопку запуска слева от неё (или Shift+Enter).

Попробуй прямо сейчас

Создай блокнот, вставь в ячейку эту строку и запусти:

print("Привет! Я готов к курсу ML")

Под ячейкой должна появиться надпись. Поздравляю — ты запустил свой первый код!

Полезно знать

  • Блокноты сохраняются автоматически на твой Google Диск.
  • Первый запуск ячейки бывает медленным (5–30 секунд) — это нормально.
  • Если что-то сломалось — меню «Среда выполнения» → «Перезапустить».
Это всё, что нужно для старта. Остальное разберём вместе на уроках.
3
GitHub — для портфолио (понадобится позже)
Не срочно к первому уроку. GitHub пригодится ближе к проектам — туда мы будем складывать готовые работы, чтобы показать родителям и добавить в портфолио. Аккаунт можно создать заранее, но это не блокирует старт.

Что это и зачем

GitHub — это место в интернете, где хранят код и проекты. К концу курса у тебя будет страница с твоими работами по ИИ — её можно показать кому угодно.

Как создать аккаунт

  1. Открой github.com и нажми Sign up.
  2. Укажи email, пароль и имя пользователя.
  3. Подтверди email по ссылке из письма (проверь и папку «Спам»).
Совет: выбери понятное имя, например ivan-petrov-ml — оно будет частью твоего портфолио.
Приглашение в общий репозиторий курса преподаватель пришлёт, когда дойдём до проектов. Тогда же разберём, как загружать туда работы — пошагово и вместе.

Что ещё понадобится по ходу курса

Веб-камера (уже на уроке 1)

На первом уроке и на занятии про нейросети мы используем камеру для распознавания объектов и эмоций. Разреши браузеру (Chrome или Edge) доступ к камере, когда он спросит.

Hugging Face (примерно 8–10 неделя)

Платформа с готовыми ИИ-моделями. Бесплатная регистрация на huggingface.co понадобится для уроков про современные модели и веб-приложения. Заранее можно не спешить.

API для LLM (блок про нейросети-помощники)

Для уроков с ChatGPT-подобными моделями преподаватель выдаст учебные ключи или бесплатные альтернативы. Платить отдельно не нужно.

Для продвинутых: локальная установка (по желанию)
Этот слайд можно пропустить. Весь курс проходит в Colab. Но если ты хочешь писать код прямо на своём компьютере (это пригодится в будущем) — вот как настроить. Если что-то не выйдет, ничего страшного: на уроках это не нужно.

Python

Скачай с python.org/downloads версию 3.11 или 3.12.

При установке на Windows обязательно поставь галочку Add Python to PATH — иначе команды не заработают в терминале.

Проверка в терминале:

python --version

VS Code + Git

Редактор кода: code.visualstudio.com. После установки добавь расширения Python и Jupyter.

Git (для работы с GitHub локально): git-scm.com/downloads. Проверка:

git --version

Чтобы коммиты подписывались твоим именем:

git config --global user.name "твой-логин" git config --global user.email "твой@email"
Застрял на установке? Сделай скриншот ошибки и пришли преподавателю — поможем. Но помни: для уроков это не обязательно, Colab всё закрывает.
