Задание Закончить регистрацию на GitHub и Hugging Face.
🐙
GitHub — github.com
  1. Открыть github.com → кнопка Sign up.
  2. Ввести email, пароль, имя пользователя (понятное, напр. ivan-petrov-ml).
  3. Подтвердить email по ссылке из письма (проверить «Спам»).
Готово, если: ученик входит и видит свою страницу профиля.
🤗
Hugging Face — huggingface.co
  1. Открыть huggingface.co → Sign Up.
  2. Ввести email, пароль, придумать username.
  3. Подтвердить почту по ссылке из письма.
Готово, если: ученик вошёл и видит главную со своим аккаунтом.
Задание Сделать свою демо-модель на Teachable Machine и сохранить, чтобы показать на уроке.
1
Открыть и создать проект
  1. Зайти на teachablemachine.withgoogle.com → Get Started.
  2. Выбрать Image Project → Standard image model.
2
Дать модели примеры (это и есть обучение!)
  1. Class 1: включить камеру и наснимать, напр. 👍 (палец вверх).
  2. Class 2: наснимать другой жест, напр. ✌️ (виктория).
  3. Чем больше примеров — тем умнее модель (30–50 кадров на класс).
3
Обучить и проверить
  1. Нажать Train Model — подождать пару секунд.
  2. Показать жест в камеру — модель угадывает в реальном времени!
  3. Нажать Export Model или сохранить ссылку для показа на уроке.
Связь с уроком 1: ты дал примеры (кадры), а машина сама нашла правило, как их различать — ровно как в игре «Угадай правило». Это твоя первая обученная модель! 🎉
